Detailed Description

template<unsigned int N, class T, class Alloc = std::allocator<T>>
class vigra::ChunkedArrayLazy< N, T, Alloc >

Implement ChunkedArray as a collection of in-memory chunks.

This optimizes over an ordinary MultiArray by allocating chunks only upon the first write. This is especially useful when only a small part of the entire array is actually needed, e.g. in a data viewer.
